  1. 用户为保证账户资金安全应注意哪些事项？

    彩民用户必须注意自己电脑中是否有键盘记录或远程控制等木马程序，使用病毒实时监控程序和网络防火墙并注意升级更新操作系统厂商所提供的安全补丁；

    • 尽量不要在公共场所（如网吧）等登录本网站；
    • 尽量在不同场合使用有所区别的密码；
    • 牢记密码，如作记录则应妥善保管；
    • 密码不得告诉他人，包括自己的亲朋好友；
    • 在登录或网上支付密码输入时，应防止左右可疑的人窥视；
    • 设置密码时不要选用您的身份证、生日、电话、门牌、吉祥、重复或连续等易被他人破译的数字。建议选用既不易被他人猜到，又方便记忆的密码；
    • 发现泄密的危险时，及时更换密码；不定期更换密码。
  2. 为什么登录后长时间不登录，要重新登录？
    • 在用户登录后，长时间不操作电脑，系统就认为用户离开自己的电脑了。为了避免别人趁机使用您的账户投注和保护彩民用户账户的安全，系统有要求用户重新登录的设置。
  3. 为什么用户不能更改认证时提交的身份证号码？
    • 用户进行投注时，身份证是保障自身利益的唯一凭证。一经注册，用户本人无法直接修改，可以联系平台工作人员修改个人信息。之所以不允许用户直接更改，主要是担心用户的用户名和密码由于个人原因泄漏，致使个人信息被他人更改。
  4. 怎么保护中奖者的个人信息？
    • 您的个人信息是保密的，福彩娱乐不会公布您的信息，请您放心购买。
  5. 如何补全及验证联系电话？
    • 登录百度彩票网站，在帐号信息中点击完善信息按钮进入补全信息界面，输入手机号并点击获取验证码，系统将自动向您指定的手机号发送验证码，输入验证码后提交保存即可完成手机号的信息补全及验证，输入手机号在未验证直接提交保存，该手机号将被记做未验证状态。
    • 如果长时间未收到验证短信，请检查联系电话并重新获取验证码（验证码获取间隔为1分钟，同IP下1天之内允许获取3次验证码，如果超出限制，请24小时后再次验证）。
  6. 如何修改绑定的联系电话？
    • 登录后进入首页，点击“我的彩票”。
    • 在“我的彩票”页面左侧“信息管理”中选择“认证资料”。
    • 用户输入新的手机号码。
    • 提交信息，修改手机成功，系统提示成功信息。
  7. 为什么要补全验证信息？
    • 当您中奖后，系统将向您的电子邮箱及联系电话发送告知信息。若中出大奖则会与您核实身份信息。为确保您能快速提款及中奖信息传达准确，请您务必填写真实的身份信息、联系电话及电子邮箱。
  8. 什么是资金密码？
    • 资金密码是保障您资金账户安全的一道防线，能有效的降低因账户被盗而引发的经济损失。此功能不强制使用，您可自行设定资金密码甚至关闭支付密码功能。为了您的账户安全，建议您使用此功能。
    • 请确保登录密码与支付密码不同，密码长度6-20位。
  9. 忘记资金密码？

    你可以联系我们的工作人员进行修改，步骤如下：

    • 联系客服人员（可通过电话，邮件等途径），说明来意；
    • 告知帐号及身份信息。
    • 客服确认身份后设置新资金密码。
